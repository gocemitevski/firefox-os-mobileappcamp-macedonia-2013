Firefox OS

Everything in the OS is open web technology and standard proposals

Firefox OS

  • The only truly open-source mobile operating system out there
  • Does not include proprietary technologies
  • Both the operating system and apps for it are written in HTML5. The hardware access happens via a Linux core
  • All you need to know to write an app for Firefox OS is to know HTML5
  • If you develop for the Web, you are developing for Firefox OS
  • It is aimed at markets that don't have access to the current leading operating systems
